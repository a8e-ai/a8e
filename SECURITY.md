# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in a8e, please report it responsibly
by emailing **security@a8e.ai**.

Please do **not** open a public GitHub issue for security vulnerabilities.

We will acknowledge your report within 48 hours and provide an estimated
timeline for a fix.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
