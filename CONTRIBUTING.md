# Contributing to a8e (Articulate)

Thank you for your interest in contributing! a8e is a hard-fork of
[Goose](https://github.com/block/goose), rebuilt with a focus on developer
sovereignty, zero telemetry, and local-first AI.

## Getting Started

1. Fork the repo and clone locally
2. Install Rust (stable channel)
3. Build: `cargo build`
4. Test: `cargo test`

## Making Changes

- Create a branch from `main`
- Write clear commit messages
- Add tests where applicable
- Run `cargo clippy` and `cargo fmt` before submitting

## Pull Requests

- Open a PR against `main`
- Describe what changed and why
- Reference any related issues
